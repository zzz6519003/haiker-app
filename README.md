[Haiker](https://haiker.app/) is a beautiful Hacker News App. Available on iOS and Android.

> NOTE: Haiker is not an open source project. This repo is for maintaining the issue and feedback.

## Download

Available on [App Store](https://apps.apple.com/app/id6762614600) and [Google Play Store](https://play.google.com/store/apps/details?id=org.randynamic.haiker)

## Features

- Readable comment thread design
- Article reader mode
- Upvote, Favorite, Reply stories and comments
- Check new replies
- Auto Translation. Perfect for non-english speaker
